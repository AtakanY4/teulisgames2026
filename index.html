SYNCOPE

Biofeedback-driven gameplay

SYNCOPE is a biofeedback-driven action game that transforms the player's real-world heart rate into a core gameplay mechanic. By integrating an Arduino-based pulse sensor directly with the game engine, the game adapts enemy behavior, wave intensity, and tempo in real time.

Hardware meets game design

When the player's heart rate rises, encounters become faster and more aggressive; when it slows, gameplay shifts into a more controlled and tactical flow. This creates a continuous feedback loop between the player's physical state and the digital world.
